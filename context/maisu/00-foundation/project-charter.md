## Objective
Build a tourism AI MVP for Bilbao that provides reliable, local-first recommendations and cultural context.

## MVP In Scope
- Place recommendations (RAG-backed)
- History/culture Q&A
- Basic itinerary suggestions

## Out of Scope (MVP)
- Full admin panel
- Multi-city support
- Advanced user personalization

## Success Metrics (initial)
- Response latency: p95 <= 8s for standard queries
- Quality/relevance: >= 80% helpful rating in manual weekly review sample
- Cost per conversation: within defined MVP budget ceiling (tracked weekly)
